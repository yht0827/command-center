# 공통 용어 사전

> 도메인을 가리지 않고 워크스페이스 전체에서 쓰이는 용어입니다.
> 도메인별 용어는 `wiki/{도메인}/glossary.md`를 참조하세요.

| 용어 | 설명 |
|------|------|
| Command Center | 이 워크스페이스. PO/디자이너/FE/BE가 공용으로 사용하는 AI 작업 공간 |
| GHE | GitHub Enterprise (github.com). 사내 코드 호스팅 |
| PRD | Product Requirements Document. 제품 요구사항 문서 |
| ontology/ | 도메인 개념·관계·코드 위치를 구조화한 탐색 맵 (YAML). Claude가 코드 작업 전 참조 |
| wiki/ | 도메인별 비즈니스 정책·의사결정·시나리오를 정리하는 디렉토리 (Markdown) |
| projects/ | GHE에서 clone한 실제 코드 레포지토리가 위치하는 디렉토리 |
